🦎 SAPO Widget - Parrilla Embebible

Ejemplo de uso del widget JavaScript

📖 Instrucciones de uso

1️⃣ Método simple (un contenedor)

Añade este código HTML donde quieras mostrar la parrilla:

<div id="sapo-widget" data-station="TU_EMISORA"></div> <script src="https://tudominio.com/sapo/sapo-widget.js"></script>

2️⃣ Método múltiple (varios contenedores)

Puedes tener varias parrillas en la misma página:

<div class="sapo-widget" data-station="emisora1"></div> <div class="sapo-widget" data-station="emisora2"></div> <script src="https://tudominio.com/sapo/sapo-widget.js"></script>

3️⃣ Configuración avanzada

Puedes personalizar la URL base de la API:

<div id="sapo-widget" data-station="TU_EMISORA" data-api-url="https://tudominio.com/sapo"></div> <script src="https://tudominio.com/sapo/sapo-widget.js"></script>
💡 Nota: Reemplaza TU_EMISORA con el nombre de usuario de tu emisora en SAPO.

✨ Características

  • Sin iframe: Se integra nativamente en tu página
  • Responsive: Se adapta a dispositivos móviles
  • Caché inteligente: Actualización cada 5 minutos
  • Colores personalizables: Usa la configuración de tu emisora
  • CORS habilitado: Funciona en cualquier dominio
  • Lightweight: Solo ~12KB minificado

🎨 Personalización

Los colores y estilos se configuran desde el panel de SAPO:

  • Color principal del widget
  • Color de fondo
  • Estilo (moderno, clásico, compacto, minimalista)
  • Tamaño de fuente (pequeño, mediano, grande)

🔌 API REST

También puedes acceder directamente a la API JSON:

GET https://tudominio.com/sapo/api_schedule.php?station=TU_EMISORA

La respuesta incluye:

  • Programación completa de la semana
  • Información de cada programa (título, descripción, imagen, redes sociales)
  • Configuración de colores y estilos
  • Datos de la emisora

🚀 Caché y rendimiento

  • Caché del servidor: 5 minutos (header X-Cache: HIT/MISS)
  • Caché del navegador: 5 minutos (Cache-Control)
  • Actualización automática: El widget se actualiza solo
